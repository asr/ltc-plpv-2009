------------------------------------------------------------------------------
Code accompanying the paper "Embedding a Logical Theory of Constructions
in Agda" by Ana Bove, Peter Dybjer, and Andrés Sicard-Ramírez (PLPV'09).
------------------------------------------------------------------------------

The code presented here does not match the paper exactly. The code has
been tested with Agda 2.3.0

Agda is available at http://wiki.portal.chalmers.se/agda/pmwiki.php

An overview of the code is given in the file src/LTC.agda.

To type check all the code run the following command:
  $ agda -isrc src/LTC.agda
